## 注意
1. 当你不需要使用该插件时，建议把开关关上，以免对页面正常浏览造成影响。
2. 该插件只会修改XMLHTTPRequest对象里的返回值，你在chrome的devtools的network里看到的请求返回结果不会有任何变化。
3. 在安装该插件前就已经打开着的页面上，该插件无法生效，需要刷新页面（或者第一次安装完直接重启浏览器吧）。
